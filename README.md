# Analisis Exploratorio de Datos IDEAE 2026

Material del curso Introduccion al Diseno Experimental y Analisis de Datos (IDEAE),
Postgrado de Inmunologia, IVIC, Semestre I - 2026.

## Objetivo

Este repositorio organiza ejemplos practicos para aprendizaje de analisis exploratorio
de datos en R, con enfasis en:

- Carga e inspeccion inicial de datos.
- Limpieza y filtrado de observaciones.
- Visualizacion estadistica base y con ggplot2.


## Contenido actual

- [AED_covidtesting.r](AED_covidtesting.r): guia paso a paso de analisis exploratorio usando el dataset
	covid_testing del paquete medicaldata.

## Requisitos

- R (version recomendada: 4.2 o superior).
- Paquetes usados en el script principal:
	- medicaldata
	- ggplot2
	- skimr

## Instalacion de R y RStudio

Guia paso a paso disponible en:
[INSTALACION_R_RSTUDIO.md](INSTALACION_R_RSTUDIO.md)

## Ejecucion rapida

1. Abrir R o RStudio en la carpeta del repositorio.
2. Instalar dependencias si aun no estan instaladas.
3. Ejecutar el script:

	 Rscript AED_covidtesting.r

## Nota de uso

El script esta orientado a docencia. Algunas instrucciones de instalacion de paquetes
estan incluidas dentro del codigo para facilitar su primera ejecucion.
